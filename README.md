# RiotAPI

Fetching data from Riot's API

## Requirements

-  Axios `npm i axios`
-  dotenv `npm i dotenv`
-  ejs `npm i ejs`
-  Express `npm i express`
-  Morgan `npm i morgan`
-  Nodemon `npm i nodemon`

-  Rename the file ".env_sample" to ".env" and put your APIKEY in the placeholder.
